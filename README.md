# OpenClaw & 有道龙虾 安装配置指南

一个简单的小程序，教你如何安装和配置 OpenClaw 和有道龙虾 (LobsterAI)。

## 使用方法

直接在浏览器中打开 `index.html` 文件即可查看教程。

## 包含内容

### OpenClaw 篇
- Docker 部署教程
- 配置文件说明
- 大模型配置 (OpenAI, Claude, 阿里通义等)
- 飞书/钉钉集成

### 有道龙虾 篇
- Windows 一键安装
- 大模型配置
- 飞书/钉钉集成
- 常见问题解答

## 预览

双击打开 `index.html` 即可在浏览器中查看交互式教程。

## 技术栈

- 纯 HTML/CSS/JavaScript
- 响应式设计，支持手机/平板/电脑
- 现代简约 UI 设计
