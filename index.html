🐙 什么是 OpenClaw？

OpenClaw 是一个开源的 AI Agent 框架，可以本地部署，支持多种大语言模型连接。可通过 Docker 快速部署，支持飞书、钉钉等 IM 集成。

Docker 部署
多模型支持
飞书/钉钉集成
免费开源
环境要求：Docker Desktop (Windows/Mac) 或 Docker (Linux)，建议 4核8G 以上内存

⚡ 选择安装方式

⚡ 一键脚本 (推荐)

Linux/Mac 一行命令自动安装，最简单

🐳 Docker 部署

Windows/Mac/Linux 通用，需要配置

1

运行一键安装脚本

在终端中执行以下命令：

curl -sL https://dl.openclaw.com/install.sh | sh

或使用 wget：

wget -qO- https://dl.openclaw.com/install.sh | sh
2

按提示配置

脚本会自动提示你输入：

  • API Key（选择模型服务商）
  • 飞书/钉钉配置（可选）
  • 管理员账号密码
3

访问管理界面

安装完成后，打开浏览器访问 http://localhost:8080

🎉 完成！ 默认管理员账号 admin/admin123
1

安装 Docker Desktop

Windows/Mac：下载 Docker Desktop

Linux：curl -sL get.docker.com | sh

2

生成配置文件

在下方选择你的模型，填写 API Key，自动生成配置：

⚙️ 配置生成器

server:
  host: 0.0.0.0
  port: 8080

llm:
  provider: openai
  api_key: your-api-key-here
  model: gpt-4o-mini
3

保存并运行

1. 新建文件夹 openclaw

2. 创建 config.yaml，粘贴上方配置

docker run -d \
  --name openclaw \
  -p 8080:8080 \
  -v $(pwd)/config.yaml:/app/config.yaml \
  justlovemaki/openclaw:latest
4

访问 http://localhost:8080

默认账号：admin / admin123

🦞 什么是有道龙虾？

有道龙虾 (LobsterAI) 是网易有道推出的 AI Agent 产品，开箱即用，图形化界面友好，一键安装无需配置。

一键安装
图形界面
多模型支持
飞书/钉钉集成

🚀 Windows 安装

1

下载安装包

从官网或技术博客获取下载链接

2

运行安装程序

双击安装包，建议安装到 D:\LobsterAI

3

启动服务

开始菜单点击"LobsterAI"启动

4

访问 http://localhost:8765

首次访问设置管理员账号

🎉 安装完成！ 接下来配置 AI 模型

🤖 配置大模型

1

进入设置

点击左下角"设置"图标

2

选择模型

支持：OpenAI、Claude、通义千问、DeepSeek、Moonshot 等

3

填写 API Key

粘贴 API Key，选择默认模型

📱 集成飞书/钉钉

飞书

设置 → 集成 → 飞书 → 填写 App ID/Secret → 开启

钉钉

设置 → 集成 → 钉钉 → 填写 AgentId/Secret → 开启

💡 常见问题

无法访问 localhost:8765？

检查防火墙，确保端口未被占用

API 调用失败？

确认 API Key 正确，网络能访问服务商

如何更新？

下载新版本覆盖安装，或在设置中检查更新
